  ----------------------- {n : n}
  x : R
  ----------------------- {x : x, n : n}
  for i in seq(0, n - n + n / 1):
    --------------------- {x : x ∨ -1.0, n : n, i : i}
    x = -1.0
    --------------------- {x : -1.0, n : n, i : i}
